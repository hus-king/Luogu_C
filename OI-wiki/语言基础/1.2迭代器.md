# 迭代器

## 基础使用

迭代器可以用来遍历容器，例如，下面两个 for 循环的效果是一样的：

```cpp
vector<int> vec = {1, 2, 3, 4, 5};
for(int i = 0; i < vec.size(); ++i) {
    cout << vec[i] << " ";
}
for(auto it = vec.begin(); it != vec.end(); ++it) {
    cout << *it << " ";
}
```

## 相关函数

- `std::advance(it, n)` ：将迭代器 it 向前移动 n 个位置, 当双向迭代器时, n 可为负数

- `next(it, n)` ：返回一个新的迭代器, 指向 it 向前移动 n 个位置后的元素(it的位置没有变化)

- `prev(it, n)` ：返回一个新的迭代器, 指向 it 向后移动 n 个位置后的元素(it的位置没有变化)

- `begin()` ：返回指向容器第一个元素的迭代器

- `end()` ：返回指向容器最后一个元素之后位置的迭代器

- `cbegin()` ：返回指向容器第一个元素的常量迭代器

- `cend()` ：返回指向容器最后一个元素之后位置的常量迭代器

- `rbegin()` ：返回指向容器最后一个元素的反向迭代器

- `rend()` ：返回指向容器第一个元素之前位置的反向迭代器
