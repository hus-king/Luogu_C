# String

字符串是编程语言中最常用的数据类型之一，用于表示文本信息。在大多数编程语言中，字符串通常被定义为一系列字符的集合，可以包含字母、数字、符号和空格等。

## 1 声明字符串
在 C++ 中，字符串可以通过多种方式声明和初始化。以下是几种常见的方法：
```cpp
string str1; // 声明一个空字符串
string str2("Hello, World!"); // 使用字符串字面值初始化
string str3 = "Hello, C++!"; // 使用赋值语句初始化
string str4(5, 'A'); // 初始化为5个字符'A'的字符串
stirng base = "Hello";
string str5 = base + ", World!"; // 使用字符串连接初始化
string str6 = str5(base, 0, 5); // 使用子字符串初始化，结果为 "Hello"
string str7 = to_string(123); // 将整数转换为字符串
```

## 2 常用操作

C++ 的字符串类 `std::string` 提供了丰富的成员函数和操作

### 2.1 获取长度
```cpp
size_t length = str.length(); // 获取字符串长度
size_t size = str.size(); // 获取字符串大小
```

### 2.2 访问字符
```cpp
char ch = str.at(0); // 访问第一个字符，带范围检查
char ch2 = str[1]; // 访问第二个字符，不带范围检查
```

### 2.3 修改字符串
```cpp
str.append(" C++"); // 在字符串末尾添加内容
str.insert(5, ", Beautiful"); // 在指定位置插入内容
str.erase(5, 10); // 删除从位置5开始的10个字符
str.replace(0, 5, "Hi"); // 替换从位置0开始的5个字符为 "Hi"
```

### 2.4 寻找某字符（串）第一次出现的位置

`find(substring, pos)` ： 从位置 pos 开始(若不传参数则从头开始)查找子字符串 substring，返回其第一次出现的位置索引，若未找到则返回 `string::npos`。

```cpp
stirng str = "Hus_King";
string a = "King";
size_t pos = str.find(a); // pos = 4
```

### 2.5 截取子串

`substr(pos, len)` ： 从位置 pos 开始截取长度为 len 的子字符串，若不传 len 则截取到字符串末尾。

```cpp
string str = "Hus_King";
string sub1 = str.substr(0, 3); // sub1 = "Hus"
string sub2 = str.substr(4); // sub2 = "King"
```

### 2.6 插入/删除字符（串）

`insert(index,count,ch)`和`insert(index, str)` ： 在索引 index 处插入 count 个字符 ch 或字符串 str。

`erase(index, count)` ： 从索引 index 处删除 count 个字符。(若不传 count 则删除到字符串末尾)

```cpp
string str = "HusKing";
str.insert(3, "_"); // str = "Hus_King"
str.erase(3, 1); // str = "HusKing"
```

### 2.7 替换字符（串）

`replace(index, count, str)` 和 `replace(first, last, str)` ： 将从索引 index 开始的 count 个字符或迭代器区间 [first, last) 替换为字符串 str。

```cpp
string str = "Hus_King";
str.replace(3, 1, ""); // str = "HusKing"
str.replace(0, 3, "King"); // str = "King_King"
str.replace(str.begin(), str.begin() + 4, "Hus"); // str = "Hus_King"
```

### 2.8 字符串转换

`to_string(value)` ： 将数值类型转换为字符串。
`stoi(str)`、`stol(str)`、`stoll(str)` ： 将字符串转换为整数类型。
`stof(str)`、`stod(str)`、`stold(str)` ： 将字符串转换为浮点数类型。
```cpp
string str1 = to_string(123); // str1 = "123"
int num1 = stoi("456"); // num1 = 456
long num2 = stol("7890123456"); // num2 = 7890123456
float f = stof("3.14"); // f = 3.14
```
### 2.9 字符串比较

`compare(str)` ： 比较当前字符串与字符串 str 的大小，返回一个整数值：若当前字符串小于 str 则返回负值，等于则返回0，大于则返回正值。

```cpp
string str1 = "apple";
string str2 = "banana";
int result = str1.compare(str2); // result < 0，因为 "apple" 小于 "banana"
```
