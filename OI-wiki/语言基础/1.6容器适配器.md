# 容器适配器

## 1 stack

栈是一种先进后出（LIFO）的数据结构，支持在一端插入和删除元素。

### 1.1 定义

```cpp
std::stack<TypeName> s;  // 使用默认底层容器 deque，数据类型为 TypeName
std::stack<TypeName, Container> s;  // 使用 Container 作为底层容器
std::stack<TypeName> s2(s1);        // 将 s1 复制一份用于构造 s2
```

### 1.2 常用操作

1. `push(value)` ： 将元素 value 添加到栈顶

2. `pop()` ： 删除栈顶元素

3. `top()` ： 返回栈顶元素的引用

4. `empty()` ： 判断栈是否为空，返回布尔值

5. `size()` ： 返回栈中元素的个数

## 2 queue

队列是一种先进先出（FIFO）的数据结构，支持在一端插入元素，在另一端删除元素。

### 2.1 定义

```cpp
std::queue<TypeName> q;  // 使用默认底层容器 deque，数据类型为 TypeName
std::queue<TypeName, Container> q;  // 使用 Container 作为底层容器

std::queue<TypeName> q2(q1);  // 将 s1 复制一份用于构造 q2
```

### 2.2 常用操作

1. `push(value)` ： 将元素 value 添加到队列尾部

2. `pop()` ： 删除队列头部元素

3. `front()` ： 返回队列头部元素的引用

4. `back()` ： 返回队列尾部元素的引用

5. `empty()` ： 判断队列是否为空，返回布尔值

6. `size()` ： 返回队列中元素的个数

## 3 priority_queue

优先队列是一种特殊的队列，支持按照优先级顺序访问元素，默认情况下是大顶堆,即优先级高的元素先出队。

### 3.1 定义

```cpp
std::priority_queue<TypeName> pq;  // 使用默认底层容器 vector，数据类型为 TypeName，默认大顶堆
std::priority_queue<TypeName, Container, Comparator> pq;  // 使用 Container 作为底层容器，Comparator 定义优先级规则
std::priority_queue<TypeName> pq2(pq1);  // 将 pq1 复制一份用于构造 pq2
```

### 3.2 自定义比较函数

```cpp
struct my_compare {
  bool operator()(const TypeName &a, const TypeName &b) const {
    return a > b;  // 定义小顶堆
  }
};

std::priority_queue<TypeName, std::vector<TypeName>, my_compare> pq;  // 使用自定义比较函数的小顶堆
```

### 3.3 成员函数

1. `push(value)` ： 将元素 value 添加到优先队列中

2. `pop()` ： 删除优先队列中优先级最高的元素

3. `top()` ： 返回优先队列中优先级最高的元素的引用

4. `empty()` ： 判断优先队列是否为空，返回布尔值

5. `size()` ： 返回优先队列中元素的个数
